# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good intent.
- Disagreement is fine; disrespect is not.
- No harassment, slurs, doxxing, sexualized content, or sustained personal attacks.
- We support contributors writing in Arabic or English. Neither is "more correct."

## Enforcement

Report incidents to `osama.aalam@gmail.com` with subject `CONDUCT: hakim-vision`. Reports are reviewed privately. Possible outcomes range from a warning to a permanent ban from the project's spaces (GitHub, Discord).

## Scope

This Code applies to all project spaces — repository activity, issues, PRs, Discord, public talks where the project is represented — and to private communication when a project participant is using their project identity.
